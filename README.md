# 🚀 Advanced Engineering Calculator

![Python](https://img.shields.io/badge/Python-3.x-blue?style=for-the-badge&logo=python)
![PyQt6](https://img.shields.io/badge/PyQt6-GUI-green?style=for-the-badge)
![License](https://img.shields.io/badge/License-MIT-yellow?style=for-the-badge)
![Status](https://img.shields.io/badge/Status-Active-success?style=for-the-badge)

A modern and feature-rich **Advanced Engineering Calculator** developed using **Python** and **PyQt6**. The application combines standard arithmetic operations, scientific calculations, memory management, theme customization, and calculation history in a clean and professional desktop interface.

---

## 🌟 Features

### 🔢 Standard Calculator

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Percentage (%)
- Parentheses Support

### 🧮 Scientific Functions

- Sine (sin)
- Cosine (cos)
- Tangent (tan)
- Logarithm (log)
- Natural Logarithm (ln)
- Square Root (√)
- Power Functions (x², x³)
- Factorial (!)

### 📌 Mathematical Constants

- π (Pi)
- e (Euler's Number)

### 💾 Memory Functions

| Function | Description |
|----------|-------------|
| MC | Memory Clear |
| MR | Memory Recall |
| MS | Memory Store |
| M+ | Add Result to Memory |
| M- | Subtract Result from Memory |

### 📜 Calculation History

- Records previous calculations
- Displays results instantly
- Easy review of mathematical operations

### 🎨 Theme Support

- Dark Theme
- Light Theme
- Modern Engineering UI
- Clean User Experience

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/jagannathsahoo6370/advanced-engineering-calculator.git
```

### Move into the Project Directory

```bash
cd advanced-engineering-calculator
```

### Install Dependencies

```bash
pip install PyQt6
```

### Run the Application

```bash
python calculator.py
```
